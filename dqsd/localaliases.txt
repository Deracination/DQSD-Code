// This is where user aliases can be defined.  Some examples are below.  Remove the 
// leading comment characters (//) to use these.  
//
// Format:
// <alias>|<search>|<name>|<description>
//
// Where:
//   alias  - alias you enter to invoke search
//   search - URL or search where all %s will be replaced by the search string
//   name   - [optional] name of search
//   desc   - [optional] description of search 
//
// Any %s placeholders will be replaced with the search string if any is supplied.
// E.g., if 'sfp' is defined as below, entering "sfp dqsd" will take you to
// http://sourceforge.net/projects/dqsd

// movie|ym
// sf|http://sourceforge.net/|SourceForge|SourceForge main web site
// sfp|http://sourceforge.net/projects/%s|SourceForge project page|SourceForge project page - Ex. "sfp dqsd"
// exp|run explorer /e, %s|Explorer|Explore directory with folder pane - Ex. "exp c:\dev\dqsd"
// expd|run explorer /e, "C:\Program Files\Quick Search Deskbar"
// np|run notepad
// word|run winword
// excel|run excel
