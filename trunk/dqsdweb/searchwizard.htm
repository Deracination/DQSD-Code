Dave's Quick Search Deskbar Search Wizard

[ Dave's Quick Search Deskbar ]

I find one of the biggest pains in writing a new DQSD search is finding and massaging the correct HTML FORM into well-formed XML. I finally got annoyed enough to write a little utility to do some of the work for me.

I call it Dave's Quick Search Deskbar Search Wizard.

What does it do?

In a nutshell, it creates a DQSD search from the page currently displayed in your browser.

One of the first things you typically do when creating a new search is to view the source for the page containing the FORM INPUT elements.  It's tedious.  It's hard to find the right FORM when there are multiple FORMs in the page.  It's error-prone because I have to convert the HTML to well-formed XML (and I always forget to close the INPUT elements with "/>").

How do I use it?

  1. Go to the web page with the search.
  2. Click in the field you want to be receive the search string.
  3. Start up the wizard by clicking on its Internet Explorer toolbar button:
  4. You should see a dialog that looks something like the following.  Enter a name and a title.  (The name is the string that will invoke the search and will be used name the file, so keep it short and simple.)
  5. Hit OK and you'll be prompted to save the search file to your DQSD 'searches' directory.
